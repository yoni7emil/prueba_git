Atributo
Información adicional que se añade a una etiqueta para modificar su comportamiento (ej. href o src).
Box Model
Concepto de CSS que describe cómo cada elemento se rodea de padding, bordes y márgenes.
CSS
Cascading Style Sheets. Lenguaje utilizado para definir la presentación y el diseño de un documento HTML.
DOM
Document Object Model. La estructura lógica de una página web que el navegador utiliza para renderizarla.
HTML5
La versión más reciente del lenguaje de marcado estándar para la web, que incluye etiquetas semánticas y multimedia.
Responsive Design
Técnica de diseño que permite que una web se adapte automáticamente a diferentes tamaños de pantalla.
