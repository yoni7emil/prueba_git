Anatomía de un Formulario

Un formulario se compone de la etiqueta contenedora <form> y diversos elementos de entrada:

  • <label>: El texto que describe qué debe escribir el usuario.
  • <input type="text">: Para nombres o textos breves.
  • <input type="email">: Valida automáticamente que el texto tenga un formato de correo.
  • <button>: El disparador que envía la información.

Interactividad con Pseudo-clases

CSS nos permite cambiar el estilo según el estado del elemento. Los más importantes para formularios son:

:hover (Pasar el ratón)

Cambia el estilo cuando el puntero está encima. Ideal para botones.

:focus (Elemento activo)

Se activa cuando el usuario hace clic dentro de un campo de texto para escribir.

::placeholder (Texto de ayuda)

Permite dar estilo al texto gris que aparece dentro de los inputs antes de escribir.
