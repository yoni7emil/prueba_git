¿Qué es el Modelo de Caja?

En CSS, cada elemento se considera una caja rectangular. Comprender sus partes es clave para el diseño:

Margin
Border
Padding
Contenido
  • Content: El texto o imagen real.
  • Padding: Espacio interno entre el contenido y el borde.
  • Border: La línea que rodea al padding y contenido.
  • Margin: Espacio externo que separa un elemento de otros.

Selectores Básicos

Para aplicar estilos, necesitamos "apuntar" a los elementos:

Tipo Ejemplo Uso
Etiqueta p { ... } Afecta a todos los párrafos.
Clase .mi-boton { ... } Se usa con el atributo class.
ID #unico { ... } Para un solo elemento específico.
