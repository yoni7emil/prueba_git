¿Qué es HTML5?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. No es un lenguaje de programación, sino que sirve para dar **estructura** al contenido.

Conceptos Clave:

  • Tags (Etiquetas): Los ladrillos de la web (ej: <p>).
  • Atributos: Información extra dentro de las etiquetas.
  • Semántica: Usar etiquetas que tengan significado (como <article>).

💡 Dato UTN: HTML5 es la versión más reciente y permite incluir video, audio y gráficos de forma nativa.
